# 达芬奇 DaVinci Resolve

### 🎬达芬奇常用快捷键🚀

📥 媒体导入与管理：

Ctrl/Cmd + I：快速导入媒体文件

Ctrl/Cmd + Shift + I：批量导入文件夹

Ctrl/Cmd + D：删除选定的媒体

Ctrl/Cmd + B：切换到缩略图视图

Ctrl/Cmd + G：切换到列表视图

🕹️ 时间线导航：

Space：播放/暂停剪辑

K：停止播放

L：向前播放

J：向后播放

Shift + K：逆向播放

Shift + L：加速向前播放

Shift + J：加速向后播放

Home：跳转到时间线开头

End：跳转到时间线结尾

🔪 剪辑操作：

B：选择工具

N：剪辑工具

Ctrl/Cmd + C：复制选定的剪辑

Ctrl/Cmd + V：粘贴剪辑

Ctrl/Cmd + X：剪切选定的剪辑

Ctrl/Cmd + Z：撤销上一步操作

Ctrl/Cmd + Shift + Z：恢复上一步操作

Ctrl/Cmd + S：保存项目

🖌️ 编辑操作：

Ctrl/Cmd + D：分割选定的剪辑

Ctrl/Cmd + T：切换剪辑

Ctrl/Cmd + R：替换选定的剪辑

Ctrl/Cmd + M：添加标记

Ctrl/Cmd + Shift + M：删除标记

Ctrl/Cmd + Shift + D：删除选定的剪辑
